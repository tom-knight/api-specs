/orders/{order_id}

Updates an Order.

To add a product to an existing order, dont include id in the body. Include product_options if adding a product with variants.

To update a product in an order, include id in the body. The body should only contain the fields that need to be updated. Those fields that are omitted will not be changed.

To remove a product from an order, set that products quantity to 0.

To learn more about creating or updating orders, see Orders Overview.

Authorization

apiKey - X-Auth-Token
